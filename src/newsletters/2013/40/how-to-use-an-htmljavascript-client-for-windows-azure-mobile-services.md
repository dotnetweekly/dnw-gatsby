This guide shows you how to perform common scenarios using an HTML/JavaScript client for Windows Azure Mobile Services. The scenarios covered include querying for data, inserting, updating, and deleting data, authenticating users, and handling errors. If you are new to Mobile Services, you should consider first completing the Mobile Services <a href="http://www.windowsazure.com/en-us/develop/mobile/tutorials/get-started">Windows Store JavaScript quickstart</a> or <a href="http://www.windowsazure.com/en-us/develop/mobile/tutorials/get-started-html">HTML quickstart</a>. The quickstart tutorial helps you configure your account and create your first mobile service.
